# devcontainer

## 说明

一个非常简单的开发容器示例。

1、使用自行构建的node镜像。镜像中一定要安装git，否则无法实现代码版本控制。

2、codespaces简单来说，就是远程起一个容器，依托vscode，实现远程开发。

## 使用

点击创建codespaces
